## Application description
![Homepage](./public/Game%20Hub%20Main.gif)
The applicaiton created is a game discovery applicaiton using [RAWG](rawg.io)'s API to obtain the database of games. The user is able to explore the multiple games released across multiple mainstream platforms. Each game listed in the grid shows their name, compatible platforms, their critc rating, and an emoji indcating whether the game is **Meh** *(not good)* üòë, **Good** üëçüèΩ, or **excellent** üéØ. 


Users can alter the grid of games by using the filtering feature. 
The user can filter by Genre of games,
![Filter by Genre](./public/Game%20Hub%20Genres.gif)

the platform they are released on, 
![Filter by Platform](./public/Game%20Hub%20Platform.gif)

or by keywords typed in the top search bar. 
![Filter by Keyword](./public/Game%20Hub%20Search.gif)
Users can also filter by using a combination of all three filter methods.

The User is also able to reorder the games in the grid by selecting one of the options in the drop-down menu: Relevance (default), Date added, Name, Release date, Popularity, or Average rating.
![Order By](./public/Game%20Hub%20Order%20By.gif)

The application is also responsive to multiple aspect ratios, making it compatible across various devices (e.g., cell phones, tablets, and laptops), and includes a toggle switch for a dark/light mode.
![Phone Aspect Ration, light mode](./public/Game%20Hub%20Phone%20Light%20Mode.gif)
